Conditional Expectation Function

Consider the Conditional Expectation Function (CEF), E[Yi|Xi]. If X takes on discrete values: Xi ∈ {x1, x2, ..., xm}, then

E[Yi|Xi] = E[Yi|Xi = x1] ⋅ 1{Xi = x1} + ... + E[Yi|Xi = xm] ⋅ 1{Xi = xm} where 1{Xi = xm} is a dummy variable,  = 1 when Xi = xm. Since the values of Xi are mutually exclusive there is no overlap of these dummy variables.

Note, we do not need to assume that X is a single random variable. It can be a vector of random variables that takes on discrete values.

We can re-arrange this expression using anyone of the values of X. The natural option is to choose the first, but this is arbitrary.

E[Yi|Xi] = E[Yi|Xi = x1] + (E[Yi|Xi = x2] − E[Yi|Xi = x2]) ⋅ 1{Xi = x2} + ... +(E[Yi|Xi = xm] − E[Yi|Xi = x2]) ⋅ 1{Xi = xm}

Since, E[Yi|Xi = xm] is a constant (Xi is set to a specific value), we can express the CEF as a function that is linear in parameters.

E[Yi|Xi] = β1 + β2Di2 + ... + βmDim

where Dim1{Xi = xm}.

  • female_black = female×black

  • male_black = (1-female)×black

  • female_nonblack = female×(1-black)

  • male_nonblack = (1-female)×(1-black)

. gen female_black=female*black

. gen female_nonblack=female*(1-black)

. gen male_nonblack=(1-female)*(1-black)

10. Theoretically, how would you test the following restrictions for the model below?

  1. β2 = β3

  2. β4 + β5 = 1

  3. β2 = β3 and β4 + β5 = 1

Y = β1 + β2X2 + β3X3 + β4X4 + β5X5 + ε

  1. Restrictions 1: H0 : β2 = β3 ⇒ β2 − β3 = 0. This can be written as a simple T-test (or F-tests),

S = (β̂2 − β̂3)/se(β̂2 − β̂3)

where,

se(β̂2 − β̂3) = [Var(β̂2) + Var(β̂3) − 2 ⋅ Cov(β̂2, β̂3)]1/2

Alternatively, rewrite the model, adding and subtracting β3X2 (or β2X3):

Y = β1 + (β2 − β3)X2 + β3(X2 + X3) + β4X4 + β5X5 + ε

Then test the hypothese that the coefficient on X2 is  = 0.

  1. Restrictions 1: H0 : β4 + β5 = 1. This can be written as a simple T-test,

S = (β̂4 + β̂5 − 1)/se(β̂4 + β̂5)

where,

se(β̂4 + β̂5) = [Var(β̂4) + Var(β̂5) + 2 ⋅ Cov(β̂4, β̂5)]1/2

Alternatively, rewrite the model, adding and subtracting β5X4 − X5 (or β2X3):

Y − X4 = β1 + β2X2 + β3X3 + (β4 + β5 − 1)X4 + (β5)(X5 − X4) + ε

Then test the hypothese that the coefficient on X4 is  = 0.

  1. To test both of the linear restrictions simultaneously, we would use an F-test.

Step 1: estimate the unrestricted model

Y = β1 + β2X2 + β3X3 + β4X4 + β5X5 + ε

Store the SSRU

Step 2: estimate the restricted model

(Y − X4) = γ1 + γ2(X2 + X3) + γ5(X5 − X4) + ε

Store the SSRR.

Step 3: Compute the F-statistic

F − stat = (SSRR − SSRU)/SSRU * dfU/(dfR − dfU)
